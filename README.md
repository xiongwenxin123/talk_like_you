# talk_like_you
这是一个可以私有化训练方言和模仿用户说话方式，音色，方言的应用


项目简介
本项目是一个面向方言场景的 AI 学习与应用平台，目标是教大语言模型（LLM）理解并生成方言，以及通过方言相关语音数据，让 ASR（语音识别） 和 TTS（语音合成） 能够识别、理解并说出方言。

项目的核心工作是方言数据的采集、标注与建模，围绕「方言词汇—语音—文本」三者的映射关系构建高质量数据集，并在此基础上训练/微调对应的模型。

功能概览（规划中）
方言数据采集

录制方言语音（多说话人、多场景、多情绪）
采集对应的标准汉字文本
衍生/维护方言词汇表与解释
数据标注与处理

方言语音切分与质检
语音对齐：方言音频 ↔ 音素序列 ↔ 汉字标注
生成或引入方言音素体系 / 音系描述
数据清洗、格式转换（如 JSON、CSV、Kaldi/ESPnet/Fairseq 等格式）
LLM 方言能力建设

通过方言词汇表、平行语料（普通话-方言）和会话数据，微调 LLM：
让 LLM 认识/解释方言词汇
支持方言与普通话之间的互译
支持方言风格的文本生成（聊天、故事、本地化对话等）
ASR：方言语音识别

使用方言语音 + 音素标注 + 汉字标注，训练/微调 ASR 模型
支持将方言语音转写为汉字（或方言拉丁/拼音方案）
评估识别准确率、字错率（CER）、词错率（WER）等
TTS：方言语音合成

基于方言语音 + 音素/文本标注，训练/微调方言 TTS 模型
输入汉字（或方言词汇/音素）合成对应的方言语音
支持不同方言口音、说话人风格
数据结构与流程概述
典型的数据与训练流程可以概括为：

采集
采集多条方言语音 audio.wav
为每条语音标注：
对应的汉字文本：text_zh
对应的音素序列：phoneme_seq
预处理
语音切分、去噪、采样率统一
文本规范化（数字、英文、标点处理等）
根据模型需求生成特定配置/清单文件（如 train.tsv, metadata.csv 等）
模型训练 / 微调
LLM：使用方言词汇/平行语料进行指令微调或对话微调
ASR：使用语音 + 文本/音素数据训练识别模型
TTS：使用语音 + 文本/音素数据训练合成模型
推理与应用
语音输入 → ASR → LLM 理解/对话 → 文本输出 → TTS → 方言语音回复
可以集成到 Web/移动端应用，形成完整的方言 AI 助手。
